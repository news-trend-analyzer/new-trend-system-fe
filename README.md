# News Trend System Frontend

뉴스 트렌드 시스템 프론트엔드 프로젝트

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **Framer Motion** - 애니메이션
- **RemixIcon** - 아이콘

## 프로젝트 구조

```
src/
├── types/              # TypeScript 타입 정의
├── constants/          # 상수 데이터
├── components/         # 컴포넌트
│   ├── layout/        # 레이아웃 컴포넌트
│   ├── trend/         # 트렌드 관련 컴포넌트
│   └── modal/         # 모달 컴포넌트
├── hooks/              # 커스텀 훅
├── App.tsx            # 메인 앱 컴포넌트
├── main.tsx           # 진입점
└── index.css          # 전역 스타일
```

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 프로덕션 미리보기

```bash
npm run preview
```
